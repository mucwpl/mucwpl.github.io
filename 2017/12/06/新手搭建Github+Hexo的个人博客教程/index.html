新手搭建Github+Hexo的个人博客教程

一、安装Node.js

直接去官网下载，各种系统和版本都有，参照自己电脑版本下载，安装一直点击next就可以了。

https://nodejs.org/en/download/

二、安装Git

去官网下载Git,安装 Git 后,打开 git bash 交互界面 , 也就是 git 命令窗口 , 以下的搭建blog所有命令操作全部在git bash进行

https://git-scm.com/download/

三、注册GitHub

这个自行注册…过程省略…

四、检查本机SSH key设置

1、检查本机的ssh key,输入以下命令

$ cd ~/. ssh

如果提示：No such file or directory ，这就说明你是第一次使用git

2、生成新的 SSH key

邮箱地址就是你注册GitHub的邮箱

注意：-C是大写的C

然后连续输入2次密码

这个密码会在你提交项目时使用 , 如果为空的话提交项目时则不用输入。这个设置是防止别人往你的项目里提交内容，最后如果没有异常信息出现, 这就说明成功设置ssh key了
