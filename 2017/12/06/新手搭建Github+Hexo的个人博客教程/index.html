3、添加ssh key 到GitHub

  • 打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件。此文件里面内容为上面步骤生成的ssh key秘钥。如果没显示这个文件，你需要设置电脑显示你本地隐藏文件。准确的复制这个文件的内容，才能保证设置的成功。
  • 然后登陆你的github
  • 点击你github右上角头像处的下拉列表 Settings—>SSH and GPG keys —> 右上角 New SSH key
  • 把刚才本地生成的密钥复制到里面（key文本框中）， 点击 add key 就ok了

五、使用GitHub搭建个人blog

​ 1、登录你的GitHub后，点击页面右下角的New Repository

​ 2、填写项目信息，Repository name：userName.github.io
​ 注意：userName为你的用户名

​ 注意：GithubPages的Repository名字是特定（两个要一致相同）的，比如你的Github账号是andy，那么你的GithubPages Repository（GitHub仓库）名字就是andy.github.io。

每个帐号只能有一个Repository来存放个人主页，而且仓库的名字必须是username.github.io，这是特殊的命名约定。设置成功后可以通过http://username.github.io 来访问你的个人博客主页。然后点击Create Repository完成创建
